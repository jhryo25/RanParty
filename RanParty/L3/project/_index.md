# L3/project — 当前项目追踪

> 此处追踪你的活跃项目。

## 使用方式

1. 为每个项目创建子目录
2. 统一入口文件为 `project.md`
3. 每次收工时更新进度状态和下一步
4. 项目完结时归档到 `../history/`

## project.md 模板

```markdown
# 项目名 — 项目档案
> 版本：x.y | 最后更新：YYYY-MM-DD

## 项目目的
<一句话说明>

## 项目周期
- 开始：YYYY-MM-DD
- 当前阶段：<描述>

## 进度状态
| 模块 | 状态 | 说明 |
|------|------|------|

## 下一步
<待办列表>
```
