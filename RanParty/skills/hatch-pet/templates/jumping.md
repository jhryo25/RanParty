# jumping.md — Jumping Animation (Row 4, 5 frames)

{{system}}

Generate the JUMPING animation row. Character does a happy jump.

Requirements:
- 5-frame horizontal strip, total 960x208 pixels.
- Frame 1: crouch preparation (knees bend, body lowers).
- Frame 2: launch upward (legs extend, body rises ~20px above baseline).
- Frame 3: peak of jump (arms up, ears up, maximum height ~30px above baseline).
- Frame 4: falling (body descends, arms start lowering).
- Frame 5: landing (knees absorb impact, return to baseline).
- Happy/excited expression throughout.
- 5-frame loop, plays once then returns to idle.
